## 作品の音楽的ポジション

| 軸 | 方向性 |
|---|---|
| ジャンル | オーケストラル・ファンタジー × アンビエント |
| トーン | 「ノスタルジック」「哀愁」「希望の予感」── 壮大だが押しつけがましくない |
| 和洋 | 主軸は西洋弦楽。尺八・篠笛等の和楽器を点描的に使用（廃墟の日本的記憶として） |
| テンポ感 | 基本的にゆっくり。感情に引っ張られるリズム。固定BPMに縛られない |

## メインテーマの設計

### ライトモチーフ（4音の主旋律）

```
ド → ラ → ファ → ソ（Aマイナー基調）
```

この4音が作品全体に渡って形を変えながら登場する：

| シーン | 変奏形態 |
|---|---|
| タイトル | ピアノ + ヴァイオリンのユニゾン |
| 邂逅（scene_01） | ヴァイオリンのみ、pppp、一瞬 |
| 「名前のない歌」との関係 | シルアの歌のメロディーは、このライトモチーフから派生している（視聴者は気づかない） |
| ティザー・クライマックス | フルオーケストラ + コーラス |
| エピローグ（第11章） | ピアノのみ、余韻 |

## 「名前のない歌」の音楽的仕様

（詳細は `song_lyrics.md` 参照）

- **形式**: アカペラ（パイロット版）/ 弦楽伴奏版（本編）
- **スケール**: Aマイナーまたはドリアン
- **BPM**: 自由（歌に引っ張られる）
- **歌詞言語**: 架空言語（エルフ語）
- **歌唱スタイル**: 囁き〜小声。決して力強くしない
- **メインテーマとの関係**: 歌のメロディーはメインテーマのライトモチーフを逆行・変奏したもの

## SE（効果音）定義

| ID | 音 | タイミング | 備考 |
|---|---|---|---|
| SE_01 | 金属が風に揺れる音 | ルカの日常シーン | 廃墟の環境音 |
| SE_02 | 遠くから聞こえる不明瞭な音 | 森の奥へ進む直前 | シルアの存在の予感 |
| SE_03 | 石が浮く、かすかな音 | 最初の石が浮く瞬間 | マジカルだが静か |
| SE_04 | 水滴が宙に漂う音 | 水滴の浮遊 | 音として聴こえるかどうかぎりぎりのレベル |
| SE_05 | 緑の光が舞う音 | 光の粒子登場 | クリスタルのような、柔らかい |
| SE_06 | 管弦楽の一撃 | 第10章フラッシュ冒頭 | BGM_05と同期 |
| SE_07 | 遠くから聞こえる「歌」の残響 | フラッシュ終了後 | シルアの歌声の残像 |

## 外注発注時のチェックリスト

作曲家に渡す素材：
- [ ] このドキュメント（music_direction.md）
- [ ] song_lyrics.md（「名前のない歌」の詞と発音ガイド）
- [ ] script_v1.md（脚本、各シーンの尺を確認させる）
- [ ] concept_one_pager.md（作品全体の感情の約束）
- [ ] 参照楽曲のリスト（Spotify/YouTubeのURLで提供）
- [ ] KV画像 5〜10枚（世界観の視覚的インプット用）

交渉時の優先順位：
1. メインテーマのデモ（30秒）を先に作ってもらい、方向性を確認
2. OKが出たら全7トラック + 「名前のない歌」伴奏版を一括発注
3. 「名前のない歌」の歌唱は別途歌い手に発注（作曲家は旋律とコード進行を提供）
