## 2. AI生成物の権利

### 2.1 使用AIツール別チェック（利用規約確認必須）

| ツール | 用途 | 商用利用 | 確認事項 |
|---|---|---|---|
| Runway Gen-3 | Img2Video | 有料プランで可 | 学習データの出典・著作権侵害責任の所在 |
| Kling AI | Img2Video | プランにより可 | 中国系。商用利用条項・出力物の排他利用権の有無 |
| Sora / OpenAI Video | Img2Video | 要確認 | API版と一般版で条件差あり |
| Midjourney | 画像生成（既存kv01〜kv18の多くがこれに該当する想定） | 有料プランで可 | v6以降は商用可。出力物の独占使用権は一部制限あり |
| Stable Diffusion（ローカル） | 画像生成 | モデルによる | ベースモデルのライセンス（CreativeML Open RAIL-M等）確認 |
| ElevenLabs | 音声生成（参考用・本番不使用） | 有料プランで可 | 声優収録が本番のため参考止まり |
| Suno / Udio | 作曲補助 | 有料プランで可 | **本番には不使用**（作曲家発注のため） |

**判断ルール**:
- 本番素材には「商用利用可」かつ「学習データの権利クリア」を明言しているツールのみ使用
- 出力時のプロンプトとシード値を記録（後日の争点発生時の反証資料）
- 1枚のKV画像について複数ツールの出力を混ぜない（トレーサビリティ確保）

### 2.2 学習データ由来のリスク

- [ ] 実在の俳優・声優・有名人に酷似した顔が出力されていないか目視確認
- [ ] 他作品のキャラクター（版権キャラ）に酷似していないか確認
- [ ] ブランドロゴ・商標が背景に写り込んでいないか確認
- [ ] 実在の建築物（著作権保護対象となるもの）が主題になっていないか確認

**推奨対応**: 上記のいずれかに該当する可能性がある場合は、当該画像を差し替える or 手描きレタッチで十分に変形する。

### 2.3 AI生成物の著作権帰属

- 現行法（日本）ではAI生成物そのものに著作権は発生しないと解釈されるのが通説
- ただし「人間による創作的寄与」（プロンプト設計・選定・後加工）が加わった部分には著作権が発生し得る
- **対策**: 全てのAI生成画像に対して、最低限 Photoshop等での色調補正・トリミング・合成・テキスト追加等の「創作的寄与」を記録する

**ログ保管**:
- [ ] 各KV画像の「AI出力 → 人間加工」の作業ログを残す（レイヤーファイル保管）
- [ ] クレジット表記には「Artwork by: [制作者名] (AI-assisted)」と明記

## 3. 楽曲権利

### 3.1 作曲家との契約（D-10・D-11）

**契約書に含めるべき条項**:

- [ ] 著作権（作曲・編曲）の帰属 → 譲渡 or 共有 or ライセンス（買い切り or 使用許諾）
- [ ] 著作者人格権の不行使特約
- [ ] 原盤権（マスター音源の権利）の帰属 → 制作側に帰属させる買い切り契約が望ましい
- [ ] JASRAC/NexTone への登録有無（登録する場合、YouTube再生時の収益分配条件）
- [ ] 二次利用条件（続編・予告編・SNSショート・CF返礼品への転用）
- [ ] 納品形式（WAV 48kHz/24bit, ステムデータ, MIDI）
- [ ] 納期・修正回数・支払条件
- [ ] 秘密保持（NDA）

**推奨方針**:
- パイロット版段階では「買い切り（著作権譲渡）」で明確化
- ただし作曲家の経歴PR用として「Anti-Gravity Sagaの主題曲を担当」と名乗ることは許可（実績掲載権）

### 3.2 歌唱者との契約（D-09）

- [ ] 実演家の権利（録音権・送信可能化権）の許諾取得
- [ ] 実演家人格権の不行使特約
- [ ] 原盤への歌唱収録の独占的利用許諾
- [ ] 二次利用条件（予告編・SNSショート）
- [ ] 収録データの保管責任と削除要請時の対応

### 3.3 参照楽曲の扱い（注意事項）

`music_direction.md` に列挙されている参照作品（もののけ姫、フリーレン、進撃の巨人等）は **方向性の参考** として作曲家に渡すものであり、いかなる形でも **そのまま使用・サンプリング・類似すぎる模倣をしない** ことを作曲家との契約に明記する。

- [ ] 参照楽曲のメロディ・コード進行を直接引用しないこと
- [ ] 類似性が高いと疑われる場合は、独立して音楽著作権判定ソフト（例: ACRCloud 等）で事前チェック
