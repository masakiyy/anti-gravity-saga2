## キャラクター別 声質・演技ブリーフ

---

### ルカ（主人公 / 少年、推定15〜17歳）

**世界観内の立ち位置**  
廃墟街で廃品回収をしながら一人で生きる。過去に何かあった。誰とも関わらないことを選んでいる。

**声の方向性**
- 低め〜中音域の少年声。完全な大人声ではなく、まだ声変わりが落ち着いたばかりの感じ
- 感情を外に出さない。セリフは最小限で、言葉を選ぶ
- ぶっきらぼうだが、冷たくはない。不器用な優しさが滲む

**禁止事項**
- 熱血・元気系の演技
- 感情を過剰に乗せる
- アニメ的な誇張（叫び、大きなリアクション）

**サンプルセリフ（感情指示付き）**

```
「安全な場所だ。」
→ 相手を安心させようとしているが、それを表に出したくない。ぶっきらぼうに、でも確かに。

「ほっとけなかった。」
→ 自分でも理由がよく分かっていない。言い訳のように、小さく。

（フードの下、初めてシルアの横顔を正視するシーン）
→ セリフなし。息遣いと間で伝える。
```

**参照声優イメージ**（雰囲気参考、指名発注ではなく傾向として）
- 村瀬歩（15歳頃の演技感覚）
- 花江夏樹（初期の抑えた演技）
- 若手新人の場合：感情を「出さない」演技ができること

---

### シルア（ヒロイン / エルフの少女、外見上15〜16歳相当）

**世界観内の立ち位置**  
記憶を失っている。名前以外、何も覚えていない。でも、不安より好奇心が勝っている。
歌だけは「なぜか」知っている。

**声の方向性**
- 透明感・儚さが最優先。澄んだ高音域
- 「今この瞬間」にだけ生きている感じ。過去への執着がない
- 不思議そうで、でも怖がっていない。純粋な観察者
- 言葉が少し「人間と違う」雰囲気（古風、ゆっくり、少し間が長い）

**禁止事項**
- ぶりっこ・媚び系
- ロリ声の誇張
- 過度な感情表現（驚き・喜びを大げさに出す）

**サンプルセリフ（感情指示付き）**

```
「……ここは？」
→ 恐怖はない。ただ、知らない場所だと事実を確認している。

「…シルア。それだけ、覚えてる。」
→ 悲しんでいない。ただそれが事実だと言っている。少し遠い目。

「（小声で、独り言のように）…おいしそう。」
→ 人生で初めて温かいものを見た時の反応。静かな驚き。

「……いつか、また、歌える気がする。」
→ 根拠のない確信。夢うつつ。
```

**歌唱への接続**  
パイロット版では歌唱も同キャストが望ましい（声の統一感）。ただし歌唱力が必須のため、歌えない場合は歌唱専任を別途手配し、台詞と歌唱を別収録して編集でつなぐ。

**参照声優イメージ**
- 種崎敦美（フリーレン本編の淡々とした感覚）
- 悠木碧（抑えた演技時の儚さ）
- 歌唱も可能な声優・歌い手：透明感を最優先

### マーサ（脇役 / 中年女性、40〜50代）

**世界観内の立ち位置**  
長年廃墟街で食堂を続けてきた。見てきたものが多い。驚かない。でも、見捨てない。

**声の方向性**
- くたびれた温かさ。サバサバしているが情がある
- 「まあ、座んな」が全てを語る声
- 人生経験を積んだ落ち着き。若作りしない

**サンプルセリフ**

```
「怪我は？」→ 事務的に、でも本当に気にかけている
「……どこで拾ったの。」→ 驚いているが、それを表に出さない
（少女の様子を見て小さく笑う）→ セリフなし
```

**参照声優イメージ**
- 渡辺明乃、浅野まゆみ 系のベテラン声優
- 舞台俳優・ナレーターでも可（セリフ量が少ないため）

### ナレーター（ルカのモノローグ、オフ）

- ルカ役と同一キャストが自然（内面の声として）
- ただし、テイクを分けて収録する（現在進行形のセリフとは異なるトーン）
- ナレーション時は「少し遠く、回想的に」

**ナレーション全文（参考）**

```
「俺の名前はルカ。廃墟街で、使えるものを拾って生きている。」
「誰とも関わらない。誰も必要としない。それが、一番楽なやり方だった。」
「お伽話の中にしか、いないはずだった。」
「その歌を、俺は知らなかった。でも、なぜか──聞いたことがある気がした。」
```

### 「名前のない歌」歌唱（SCENE 5 / 60秒）

**重要度**: 作品の核心。キャスティングで最も慎重を要する。

**要件**
- 架空言語（エルフ語擬似言語）で歌う
- スケール: Aマイナーまたはドリアン
- BPM: 自由（固定なし、歌に引っ張られるリズム）
- 音量: pppp〜mp（力強くならない。漏れてしまった音として）

**演技要件**
- 「上手く歌おう」という意図を持たせない
- 自分が歌っていることを半分意識していない演技
- 冒頭は声にならない音から始まり、自然に歌になる

**候補ルート**
1. **シルア役声優が歌唱も担当**（理想、ただし歌唱力条件あり）
2. **声優+歌い手の分業**（声優は台詞のみ、歌唱はシルア声質に近い歌い手）
3. **歌い手をシルア声優として全収録**（声優実績がある歌い手）

**参照感覚**
- 「アシタカとサン」米良美一の歌唱 → 民族的、儚さ、力まない
- 「少年と魔法使い」（ハウルの動く城） → 囁き歌の感覚

## 発注先候補（カテゴリ別）

### A. 声優事務所（正規ルート）

- **主要事務所への持ち込み**: 企画書・脚本・世界観資料を揃えてコンタクト
- **必要書類**: concept_one_pager.md + script_v1.md + KV画像5枚
- **ネック**: 最低制作費がかかる場合あり（ギャラ + 制作費）

### B. ボイスOT（ボイスオーバータレント）/ クラウドソーシング

- **Coconala / SKIMA / ランサーズ**の「声優・ナレーター」カテゴリ
- デモ音源を複数聴いて選定
- 低予算での第一選択肢

### C. 歌い手（ニコニコ動画 / YouTube系）

- 「透明感」「囁き歌」「幻想系」タグで活動している歌い手
- シルア役または歌唱専任として
- 参考チャンネル: 幻想系・和風ファンタジー系の歌い手を中心に

### D. インディー声優 / 音声SNS発掘

- Twitter/Xの「#声優志望」「#収録依頼歓迎」タグ
- 新人発掘ルートとしてコスパが高い
- ただし、プロ品質の保証が必要なため事前デモ確認必須

## キャスティング判断基準

以下の順に優先して選定する：

1. **声質が世界観に合っている**（最優先）
2. **「出さない」演技ができる**（自然体・日常演技）
3. **過去作品デモで確認可能**
4. **予算・スケジュールが折り合う**

> 「正確に上手く演じる」よりも「世界観に溶け込む」声を優先する。
> ルカは特に、セリフの少なさと間（ま）で感情を伝えられる人材を選ぶこと。
