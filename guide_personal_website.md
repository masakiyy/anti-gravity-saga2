# 自分だけのHPを無料で作成・公開する方法

「自分のHPを持ちたい！でも費用はかけたくない」という場合でも、完全に無料でプロレベルのサイトを持つことは可能です。
その仕組みと、おすすめの方法をまとめました。

## 1. 結論：無料で作れますか？
**はい、作れます。**
以下の3つの要素をすべて無料プランで賄うことができます。

1.  **HPのデータ（HTML/CSS）**: AIと一緒に書けば無料。
2.  **土地（サーバー）**: Netlifyなどのホスティングサービスを使えば無料。
3.  **住所（ドメイン）**: サービス提供のサブドメイン（例: `site.netlify.app`）を使えば無料。

## 2. 無料のおすすめ構成（最強の組み合わせ）

今回「Anti-Gravity Saga」で実践した方法が、実は世界中のエンジニアも愛用する王道かつ無料の方法です。

| 項目 | 使うツール | 費用 | 特徴 |
| :--- | :--- | :--- | :--- |
| **作成** | AI (Gemini/Cursor) + エディタ | 0円 | 自由度無限大。好きなデザインで作れる。 |
| **公開** | **Netlify (ネットリファイ)** | 0円 | ドラッグ＆ドロップだけで公開可能。高速。 |
| **URL** | Netlify Subdomain | 0円 | `https://[好きな名前].netlify.app` になります。 |

### その他の無料ツール（ノーコード）
コードを書くのが面倒な場合は、以下のツールも無料プランがあります。
*   **STUDIO**: デザイン自由度が高い。無料版は左下にバナーが出ます。
*   **Wix / Jimdo**: テンプレートから選ぶだけ。無料版は広告が入ります。

→ **見栄えにこだわるなら、広告が入らない「AIコーディング + Netlify」が一番おすすめです。**

## 3. お金がかかるケース（有料になる場合）

あえてお金をかけるとすれば、以下のポイントです。

*   **独自ドメイン (`.com` / `.jp`)**:
    *   例: `masaki-portfolio.com`
    *   費用: 年間 1,500円 〜 3,000円程度
    *   メリット: 信頼性が上がる、URLが短くて覚えやすい。
*   **大規模なアクセス**:
    *   月間数万人が訪れるようになったり、大量の動画を配信する場合は、サーバー代が必要になることがあります。

## 4. 今後のステップ

1.  **パソコンの中にフォルダを作る**: 今回の `06_HP` のように、プロジェクトごとにフォルダを分けます。
2.  **AIに頼む**: 「私のポートフォリオサイトを作って。デザインはシンプルで黒ベースで」のように依頼します。
3.  **Netlify Dropする**: 出来上がったフォルダをドラッグ＆ドロップするだけで世界に公開完了です！

ぜひ、自分だけのオリジナルHP作りにも挑戦してみてください！
