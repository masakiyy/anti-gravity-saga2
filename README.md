# 黄昏のルカと暁のエルフ -Anti-Gravity Saga-

このフォルダには、ウェブサイトのソースコードが含まれています。

## フォルダ構成
- `index.html`: メインページ
- `images/`: 画像フォルダ（背景画、キャラクター画像など）

## 公開方法（デプロイ）

このサイトをインターネット上で公開するための、簡単な方法を2つ紹介します。

### 方法1: Netlify Drop（一番かんたん！）
登録不要で、ドラッグ＆ドロップだけで公開できます。

1. [Netlify Drop](https://app.netlify.com/drop) にアクセスします。
2. この `06_HP` フォルダごと、ブラウザの枠内にドラッグ＆ドロップします。
3. 数秒待つと、アップロードが完了し、公開用URLが発行されます。
4. そのURLにアクセスすれば、スマホやPCから誰でも見られるようになります。

### 方法2: Vercel（エンジニア向け）
GitHubアカウントをお持ちの場合は、こちらがおすすめです。

1. [Vercel](https://vercel.com/) にアクセスし、GitHubアカウントでログインします。
2. "Add New..." -> "Project" を選択します。
3. このフォルダをGitHubにアップロード（push）し、そのリポジトリを選択します。
4. "Deploy" をクリックすると自動的に公開されます。

### 注意点
- 画像ファイル（`images/`フォルダ）も必ず一緒にアップロードしてください。
- ファイル名を変更すると画像が表示されなくなる可能性があります。
